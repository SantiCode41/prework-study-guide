HTML

  • Enter your HTML notes here.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The terms "element" and "tags" are sometimes used interchangeably, however an element is everything contained between the opening and closing tags.
  • HTML stands for HyperText Markup Language.
  • The tag at the top of an html file states the document type. This is a decleration so it is not case sensitve.
  • The information in the head element is not visible to the user as it is used to tell the browser how to load the page. The only exception is the title child element which is visible in the tab bar.
  • Elements can be nested.
  • Elements that are nested inside other elements are called child elements.
  • Elements that hold other elemnts inside are called parent elements.
